Estudio indie reciente

Somos una empresa indie que crea
experiencias cortas y emocionales

CLOVERFOX nació con la idea de demostrar que los juegos no necesitan ser largos para llegar al corazón. Cada título que creamos es una pequeña obra de arte interactiva, diseñada para emocionar en pocas horas.

Nuestra Misión

Creemos que el arte de los videojuegos está en la emoción, no en las horas de juego. Nuestra misión es crear títulos concisos que dejen huella: juegos que puedas completar en una o dos sesiones pero que recuerdes durante años.

Trabajamos con un modelo innovador: cada vez que un jugador completa un juego al 100%, recibe acceso gratuito a otro juego del catálogo. Así premiamos la dedicación y aumentar la comunidad.

🎨

Arte a mano

Cada fondo y cada animación está dibujada a mano. Nuestro arte es único y lo que nos distingue de otros estudios.

🎁

Sorpresa mensual

Cada mes lanzamos contenido nuevo o sorpresas para la comunidad a través de nuestro programa de juego gratuito.

👥

Comunidad primero

Escuchamos a nuestros jugadores en Discord, redes sociales y formularios de feedback. Ellos son parte del proceso creativo.

⚙️ Tecnología

Herramientas profesionales para un equipo ágil. Así es como trabajamos día a día.

🌐

WordPress como CMS

Nuestra web corporativa y tienda funcionan sobre WordPress con WooCommerce, lo que nos permite actualizar contenido fácilmente desde el panel de administración.

🔀

Perforce / Plastic SCM

Usamos Perforce Helix Core o Plastic SCM para el control de versiones de activos binarios: texturas, modelos 3D, escenas y builds compiladas.

🖥️

Escritorios Virtuales

Algunos miembros del equipo trabajan con escritorios virtuales (VDI) para acceder a estaciones de trabajo potentes desde cualquier ubicación.

🖧

Servidor Apache + PHP + MySQL

Infraestructura LAMP clásica para el hosting web. Apache como servidor HTTP, PHP para la lógica y MySQL para la base de datos de WordPress.

💾

Backups Automáticos

Copias de seguridad diarias incrementales y semanales completas de todos los repositorios, bases de datos y activos del proyecto.

🔒

Seguridad

Acceso por VPN al servidor de desarrollo, contraseñas robustas gestionadas con un gestor de contraseñas, certificados SSL/TLS y actualizaciones constantes.
